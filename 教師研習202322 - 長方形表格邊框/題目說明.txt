教師研習202322 - 長方形表格邊框


題目說明：
輸入三個正整數，分別為row、column、grid_size
其中row為表格直向的數量、column為表格的橫向的數量、grid_size為表格大小


範例輸入一：
1 2 2
範例輸出一：
+--+--+
|  |  |
|  |  |
+--+--+


範例輸入二：
2 3 3
範例輸出二：
+---+---+---+
|   |   |   |
|   |   |   |
|   |   |   |
+---+---+---+
|   |   |   |
|   |   |   |
|   |   |   |
+---+---+---+
